# Changelog

## 2.0.0

- Migrated the project to Swift 5.
- Raised the minimum supported iOS version to 12.0.
- Added Swift Package Manager support.
- Fixed a crash when `shouldBeginGestureHandler` is not set.
- Fixed touch-ending and image-cropping force unwrap crashes.
- Improved value clamping for touch, accessibility, and programmatic updates.
- Added right-to-left layout support.
- Improved redraw and intrinsic-size invalidation when inspectable properties change.
- Updated CocoaPods metadata for the new release.
